About

What is Overkill Linux?

Overkill Linux is a Debian-based Linux distribution that uses the KDE Plasma desktop environment. It offers customized editions tailored to every user type and need -- developers, gamers, multimedia creators, scientists, crypto users, engineers, and students.

Through the Overkill System Manager, all required packages for each edition are downloaded and kernel settings are applied with a single click. The System Manager also provides ZRAM support, NVIDIA driver installation, package management, DNS configuration, browser installation, and general system administration tools.

The goal of Overkill is to provide an effortless and fast workstation setup for every user. One ISO, install your preferred edition after boot, and your system is ready.

Releases

Release History

Overkill Linux was formerly known as Linux Megamix. The project evolved to address maintenance overhead, installer limitations, and to introduce a more flexible edition system.

CURRENT

Overkill Linux 2.0 "Katana"

The Linux Megamix project was evolved due to debian-installer and squashfs copy issues, low usability, and to reduce maintenance overhead. The new version introduces greater edition flexibility -- for example, Developer and Gaming editions can now be combined into Mix editions.

  • Kernel tuning profiles updated and expanded (gaming, multimedia, AI, hardened)
  • Multiple new editions added (Lite variants, Gamebox, Mix combinations)
  • Overkill System Manager for easy system and package management via GUI
  • New KDE Plasma desktop configuration with updates and improvements
  • New Conky desktop monitor with system info, performance, storage, network, and battery display
  • Conky Manager tool for enabling/disabling the desktop monitor
LEGACY

Linux Megamix 1.0

Initial release with 12 specialized editions ecosystem. Base system synchronized across all customized flavors. Standard deployment with debian-installer.
